PipraPay - Payment Automation Platform PipraPay is a self-hosted payment automation platform by QubePlug. Host it on your own server, define your workflow, install plugins and themes, and stay in control. It’s free to use with no monthly fees. Visit https://piprapay.com/ for more info. Plugin-based architecture for gateways, modules, and tools. Theme support for admin dashboard branding. Multiple gateways (Stripe, bKash, Nagad, etc.) via plugins. Automatic transaction verification to reduce manual work. Scales to unlimited transactions (within your server capacity). Integrations with platforms like WooCommerce, WHMCS, and WordPress. ⚙️ Installation Clone this repository: git clone https://github.com/PipraPay/PipraPay-Open-Source-Web.git Prepare your environment: PHP, a database, and a web server (Apache/Nginx). Upload or place the source files on your server. Install dependencies (if any) as per your setup. Configure database credentials, domain, and environment settings.
